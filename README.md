## Setup

```sh
$ make install
$ make publish
```

## Start Even-Game

```sh
$ make even
```
```sh
Answer "yes" if the number is even, otherwise answer "no"

May I have your name? Sam
Hello, Sam!

Question: 56
Your answer: Yes
Correct!
```
